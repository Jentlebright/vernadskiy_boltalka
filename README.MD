# Тестовая итерация болталки Вернадского
Надо сначала загрузить данные, потом создать env где в правильном формате подгрузить токены

запуск
```
pip install -r requirements.txt
python main.py build-graph
python main.py index
python main.py
```

локально через Ollama (сначала скачать к себе приложение и модель)

```
ollama run llama3.2
```
